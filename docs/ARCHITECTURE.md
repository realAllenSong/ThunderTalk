## Key Design Decisions

- **PySide6 over Electron/Tauri**: Mature Qt6 bindings, single-language stack, native look
- **sherpa-onnx Python**: Battle-tested ASR with ONNX Runtime + llama.cpp built in
- **Mic on-demand only**: Microphone acquired at hotkey press, released immediately after
- **ASR in background thread**: `QThread` worker prevents UI freeze during inference
- **uv for dependency management**: Fast, reproducible installs
